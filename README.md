# IP-Profit-Tax-Calculator
Программа для расчёта налога на прибыль индивидуального предпринимателя (ИП) с учётом доходов, расходов и налоговой ставки.

# Система расчёта налога на прибыль для ИП

## Описание проекта

Данный проект представляет собой консольное приложение для автоматизированного расчёта налога на прибыль индивидуального предпринимателя (ИП).

Программа позволяет пользователю ввести основные финансовые показатели и получить точный расчёт налога с учётом выбранной системы налогообложения, а также дополнительных параметров (льготы, взносы).

---

## Назначение программы

Программа предназначена для:
- автоматизации расчёта налога,
- уменьшения вероятности ошибок при ручных вычислениях,
- демонстрации принципов налогообложения ИП.

---

## Входные данные

| Параметр | Обозначение | Тип данных | Описание | Пример |
|-------------------------|------------|------------|-----------------------------------------------|------------|
| Доход за период | income | float | Общая сумма доходов за расчётный период | 500000.00 |
| Расходы за период | expenses | float | Сумма подтверждённых расходов | 200000.00 |
| Система налогообложения | tax_type | string | Вариант: УСН 6% или УСН 15% | УСН 6% |
| Налоговые льготы | benefits | float | Сумма налоговых вычетов | 10000.00 |
| Фиксированные взносы | fees | float | Обязательные страховые взносы | 40000.00 |

## Выходные данные

| Параметр | Обозначение | Тип данных | Описание | Пример |
|--------------------------|------------|------------|-----------------------------------------|---------------|
| Налогооблагаемая база | tax_base | float | Сумма, с которой рассчитывается налог | 300000.00 |
| Сумма налога | tax | float | Налог до вычетов | 45000.00 |
| Итог к уплате | total_tax | float | Итоговая сумма налога | 0.00 |
| Система налогообложения | tax_type | string | Используемый режим | УСН 15% |
| Отчёт | report | string | Подробное описание расчёта | Налог рассчитан... |

## Функциональные требования

| Параметр (функция) | Обозначение | Тип данных | Описание | Пример |
|------------------------|--------------------|------------|--------------------------------------------|----------------------------------|
| Ввод данных | input_data() | function | Получает данные от пользователя | input() |
| Проверка данных | validate() | function | Проверяет корректность значений | income ≥ 0 |
| Расчёт прибыли | calc_profit() | float | Доход − расходы | 500000 - 200000 = 300000 |
| Налог УСН 6% | calc_tax_6() | float | 6% от дохода | 500000 × 0.06 = 30000 |
| Налог УСН 15% | calc_tax_15() | float | 15% от прибыли | 300000 × 0.15 = 45000 |
| Учёт льгот | apply_benefits() | float | Вычитание льгот | 45000 - 10000 = 35000 |
| Учёт взносов | apply_fees() | float | Вычитание взносов | 35000 - 40000 = 0 |
| Ограничение ≥ 0 | normalize() | float | Минимальное значение = 0 | -5000 → 0 |
| Формирование отчёта | generate_report() | string | Создаёт текст результата | "Итоговый налог: 0" |

| Вывод результата | output() | void | Отображает результат пользователю | print() |

---

## Логика работы программы

1. Пользователь вводит исходные данные
2. Выполняется проверка корректности ввода
3. Рассчитывается прибыль:

прибыль = доход − расходы
4. Выбирается система налогообложения:
- УСН 6% → налог = доход × 6%
- УСН 15% → налог = прибыль × 15%

5. Из налога вычитаются:
- налоговые льготы
- фиксированные взносы

6. Итоговая сумма налога не может быть меньше 0

7. Выводится результат с пояснением
