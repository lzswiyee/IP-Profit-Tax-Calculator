# IP-Profit-Tax-Calculator
Программа для расчёта налога на прибыль индивидуального предпринимателя (ИП) с учётом доходов, расходов и налоговой ставки.

1. Назначение программы

Данная программа предназначена для автоматизации процесса расчёта налога на прибыль индивидуального предпринимателя(ИП) в завиисимости от выбранной системы налогообложения.
Основная цель разработки - упростить и ускорить вычисления, минимизировать вероятность ошибок при ручном расчёте, а также предоставить пользователю понятный и прозрачны результат с пояснением.

Программа может использоваться:

·  Студентами для изучения принципов налогообложения,

·  начинающим предпринимателями,

·  для демонстрационных и учебных целей.

В рамках  проекта реализуется консольное приложение с возможностью дальнейшего расширения (например, добавление графического интерфейса или поддержки других налоговых режимов).

2. Входные данные

Программа должна принимать от пользователя набор параметров, необходимых для расчёта налога.

Входные данные:

|     |     |     |     |     |
| --- | --- | --- | --- | --- |
| №   | Параметр | Обозначение | Тип данных | Описание |
| 1   | Доходы | income | float | Общая сумма доходов за период |
| 2   | Расходы | expenses | float | Сумма подтверждённых расходов |
| 3   | Налоговая ставка | tax_rate | float | Ставка налога (%) | 6   |
| 4   | Льготы | benefits | float | Сумма налоговых льгот (если есть) |
| 5   | Период Расчёта | period | string | Отчётный период (месяц/год) |
